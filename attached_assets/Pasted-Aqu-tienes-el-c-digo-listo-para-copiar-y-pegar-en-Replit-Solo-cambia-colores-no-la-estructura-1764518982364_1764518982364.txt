Aquí tienes el código listo para **copiar y pegar en Replit**. Solo cambia colores, no la estructura de tu página.

## 1. Añade la paleta al inicio del `<style>`

Dentro de tu `index.html`, en la parte de `<style>...</style>`, pega esto **arriba de todo**:

```css
:root {
  --color-primary: #1A3E5C;    /* Azul profundo: títulos, header, footer */
  --color-secondary: #1ABC9C;  /* Verde esmeralda: iconos de éxito, acentos */
  --color-cta: #FF9900;        /* Naranja CTA principal (tipo Amazon) */
  --color-cta-hover: #E07C00;  /* Hover CTA */

  --color-bg: #F7F9FB;         /* Fondo general de la página */
  --color-surface: #FFFFFF;    /* Tarjetas, bloques principales */

  --color-text-primary: #444444;  /* Texto principal */
  --color-text-secondary: #6B7280; /* Texto secundario */
}
```

## 3. No olvides los textos que deben usar los nuevos estilos

Asegúrate de que:

- El botón de “CREAR LISTA GRATIS” tenga clase `hero-cta` o `hero-cta-primary`.  
- El botón de “Buscar regalo inteligente” (si ya lo añadiste) tenga clase `hero-cta-secondary`.  
- Las tarjetas “¿Por qué Givlyn?” sigan usando la clase `feature-card`.  
- Si usas un ícono de check, puedes envolverlo con `<span class="icon-check">✓</span>` para que tome el verde `#1ABC9C`.

Con esto, Replit solo necesita que pegues este CSS en tu `index.html`, guardes y pulses **Run**: la pantalla actual se verá con la nueva paleta azul + verde + naranja que definimos.
